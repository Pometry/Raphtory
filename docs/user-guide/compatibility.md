# Compatibility

When setting up a Raphtory environment you must consider the prerequisites of the specific version you are using and compatibilities between versions. This page provides information on the supported versions of Raphtory components and dependencies.

## Python versions

The following python versions are supported:

- 3.14
- 3.13
- 3.12
- 3.11

## Raphtory versions

When installing Raphtory you should generally use the latest version of both the base library and the user interface.
